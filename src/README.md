# Курсовой расчёт (автоматизация отчёта Word)

Проект автоматизирует:
- вычисления параметров электрических элементов;
- построение графиков с точками пересечения;
- генерацию Word-документа (`docxtpl`);
- замену формул в OMML (XML внутри DOCX).

## Запуск
```bash
pip install -r requirements.txt
python -m src.main
